# Reweighting-based Approach for Event Log Fair Pre-processing

This is the git repository for the paper *Reweighting-based Approach for Event Log Fair Pre-processing* by *Denis Kasakow, Steven Purtzel and Kate Revoredo* from *Humboldt-Universität zu Berlin*

## Development setup

- Linux
- Python 3.12.11
- Package versions in requirements.txt

## Running the Code

- `main.full_experiment` expects in the directory `eventlogs/` three eventlogs generated by Käppel et al. <https://github.com/fmoehrlein/fair_pbpm_distillation>:
  - hb_-age_+gender.xes for Hospital Billing
  - cs.xes for Cancer Screening
  - bpi_2012.xes for BPI 2012
- Running `python main.py` will produce the reweighed eventlogs in the directories `full_experiment_output_{1,2,3}/` that were used to evaluate the methodology of the paper
